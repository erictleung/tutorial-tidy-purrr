# tutorial-tidyverse

Presentation on the tidyverse in R to clean and manipulate data


## Prerequisites

- `R`
- `rmarkdown` (to compile presentation)
- `tidyverse`


## Compile Presentation

```shell
make
```


## Packages within `tidyverse`

- broom
- dplyr
- forcats
- ggplot2
- haven
- httr
- hms
- jsonlite
- lubridate
- magrittr
- modelr
- purrr
- readr
- readxl
- stringr
- tibble
- rvest
- tidyr
- xml2
